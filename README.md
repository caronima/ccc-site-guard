Daily Update Report Notifications and Security Measures.

## Description

CCC Site Guard helps you maintain a secure WordPress site by providing daily update reports and implementing essential security measures. 

This plugin monitors your WordPress core, themes, and plugins for available updates, and sends you regular email notifications with detailed reports. It also includes security features to help protect your site from common vulnerabilities and unauthorized access attempts.

### Key Features

- Daily email reports of available updates
- WordPress core, theme, and plugin update monitoring
- Security enhancement measures
- Easy-to-use admin interface
- Lightweight and performance-optimized

## Installation

1. Upload ccc-site-guard to the `/wp-content/plugins/` directory.
2. Activate the plugin through the **Plugins** menu in WordPress.
3. You will find **CCC Site Guard** menu in your WordPress admin panel.
